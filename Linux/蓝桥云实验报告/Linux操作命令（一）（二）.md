# Linux操作指令（一）

## ls

ls是用来查看目录的信息

参数	描述：

|  -a  |      –all 列出目录下的所有文件，包括以 . 开头的隐含文件      |
| :--: | :----------------------------------------------------------: |
|  -l  | 除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来 |
|  -h  | –human-readable 以容易理解的格式列出文件大小（例如 1K 234M 2G) |
|  -t  |                      以文件修改时间排序                      |

## cd

cd（change directory）：切换至指定目录

- cd ..  切换到父目录
- cd ~  切换到主目录
- cd -   回到上次的目录

## pwd

pwd（print working directory）：显示当前路径

#### 常用参数

| 参数 | 描述                                       |
| ---- | ------------------------------------------ |
| -P   | 显示实际物理路径，而非使用连接（link）路径 |
| -L   | 当目录为连接路径时，显示连接路径           |

# Linux操作命令（二）

## mkdir

make directory

> mkdir [选项] 目录  创建目录

|参数	|描述|
|--------------------|-------------------------------|
|-m --mode=模式	|设定权限<模式>|
|-p --parents	|可以是一个路径名称。若路径中的某些目录尚不存在，加上此选项后，系统将自动建立好那些尚不存在的目录，即一次可以建立多个目录|
|-v --verbose	|每次创建新目录都显示信息|

## rm 

rm是remove的缩写，删除目录或文件

> cat [选项] 文件

> rm [选项] 文件或目录

|参数	|描述|
|---------|---------|
|-b --back	|若需覆盖文件，则覆盖前先行备份|
|-f --force	|如果目标文件已经存在，不会询问而直接覆盖|
|-i --interactive	|若目标文件已经存在时，就会询问是否覆盖|
|-u --update	|若目标文件已经存在，且源文件比较新，才会更新|
|-t --target	|该选项适用于移动多个源文件到一个目录的情况，此时目标目录在前，源文件在后|

## mv

mv是移动命令，是move的缩写

> mv [选项] 源文件 目标文件

|参数	|描述|
|------------|----------------------------------|
|-t --target-directory	|指定目标目录|
|-i --interactive	|覆盖前询问（使前面的 -n 选项失效）|
|-n --no-clobber	|不要覆盖已存在的文件（使前面的 -i 选项失效）|
|-s --symbolic-link	|对源文件建立符号链接，而非复制文件|
|-f --force	|强行复制文件或目录，不论目的文件或目录是否已经存在|
|-u --update	|使用这项参数之后，只会在源文件的修改时间较目的文件更新时，或是对应的目的文件并不存在，才复制文件|

## cp 

cp用来复制文件，copy的缩写

> cp [选项] 源文件 目录

|参数	|描述|
|--------------|------------|
|-t --target-directory	|指定目标目录|
|-i --interactive	|覆盖前询问（使前面的 -n 选项失效）|
|-n --no-clobber	|不要覆盖已存在的文件（使前面的 -i 选项失效）|
|-s --symbolic-link	|对源文件建立符号链接，而非复制文件|
|-f --force	|强行复制文件或目录，不论目的文件或目录是否已经存在|
|-u --update	|使用这项参数之后，只会在源文件的修改时间较目的文件更新时，或是对应的目的文件并不存在，才复制文件|

## cat

cat 命令用来输出文件，是 concatenate 的缩写

|参数	|描述|
|-----------------|------------------|
|-A --show-all	|等价于 -vET|
|-b --number-nonblank	|对非空输出行编号|
|-e	|等价于 -vE|
|-E --show-ends	|在每行结束处显示 $|
|-n --number	|对输出的所有行编号，由 1 开始对所有输出的行数编号|
|-s --squeeze-blank	|有连续两行以上的空白行，就代换为一行的空白行|
|-t	与 -vT 等价|
|-T --show-tabs	|将跳格字符显示为 ^I|
|-u	|（被忽略）|
|-v --show-nonprinting	|使用 ^ 和 M- 引用，除了 LFD 和 TAB 之外|


## nl

nl是计算题行号，是 number of lines 的缩写

> nl [选项][文件]

|参数	|描述|
|-------------|------------------|
|-b	|指定行号指定的方式，主要有两种：|
|-b a	|表示不论是否为空行，也同样列出行号（类似 cat -n）|
|-b t	|如果有空行，空的那一行不要列出行号（默认值）|
|-n	|列出行号表示的方法，主要有三种：|
|-n ln	|行号在屏幕的最左方显示|
|-n rn	|行号在自己栏位的最右方显示，且不加 0|
|-n rz	|行号在自己栏位的最右方显示，且加 0|
|-w	|行号栏位的占用的位数|
