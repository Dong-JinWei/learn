# 二维数组

应用场景：2048游戏，俄罗斯方块，排雷  线性代数：矩阵



二维数组变量：

​    //类型名 变量名；

​    //类型名[] 变量名；      类型名[] 是一个数组类型名；

​       类型名[][] 变量名；   

 

​    二维数组对象：

```
1、 new 类型名[表达式][];

2、 new 类型名[表达式1][表达式2]
```



二维数组初始化：

```
new 类型名[][]{new 类型名[表达式]，new 类型名[表达式]，new 类型名[表达式]…}

new 类型名[][]{

{值，值，值…}，

{值，值，值…}，

  ……

}
```



三维数组：

```
类型名[][][] 变量名；

new 类型名[表达式][][];

new 类型名[表达式][表达式][];

new 类型名[表达式][表达式][表达式]
```
