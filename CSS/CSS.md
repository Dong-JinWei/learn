## 1.1、什么是CSS

Cascading Style Sheet 层叠级联样表

CSS：表现（美化网页）

字体，颜色，边距，高度，宽度，背景图片，网页定位，网页浮动……

## 1.2、发展史

CSS1.0

CSS2.0	DIV（块）+CSS，HTML与CSS结构分离的思想，网页变得简单，SEO

CSS2.1	浮动，定位

CSS3.0	圆角，阴影，动画……浏览器兼容性~
